Das angehängte Bild ist das **offizielle Firmenlogo**. Erstelle eine **exakte 1:1 SVG-Kopie** davon und integriere es in die Website:

- Headlines: Playfair Display (Serif, elegant)

### DESIGN-STIL

**Modern Clean Smooth Aesthetic:**
- Viel Weißraum - atmendes Layout
- Subtile Schatten statt harte Kanten
- Smooth Animationen (ease-out, 300-500ms)
- Große, hochwertige Bilder
- Minimalistische Icons
- Klare Hierarchie

**KEINE:**
- Überladene Layouts
- Grelle Farben außer dem Rot
- Stock-Photo-Look
- Generische Templates
- Zu viele Fonts oder Farben

1. **Pixel-Perfect** - Saubere Abstände, konsistente Größen
2. **Responsive** - Desktop, Tablet, Mobile perfekt
3. **Performance** - Lazy Loading, optimierte Assets
4. **Accessibility** - Kontraste, ARIA-Labels
5. **Smooth UX** - Keine Ruckler, flüssige Animationen

### DELIVERABLE
